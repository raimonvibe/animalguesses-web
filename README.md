# 🐾 Animal Guesses Web App 🐾  
🔗 [Visit the App](https://animalguesses.raimonvibe.com/)

Welcome to **Animal Guesses**, a fun and interactive animal guessing game for all ages! 🎮  
Try to match animal sounds with the correct pictures 🐶🦁🐷 and see how many you can guess correctly! 🧠✨

---

## 🧩 Features

- 🎵 **Audio-based guessing** – Hear the sound, and guess the animal!
- 🖼️ **Cute animal images** – Match the correct image to the sound.
- ✅ **Simple yes/no interaction** – Super accessible and easy to play.
- 📱 **Mobile-friendly** – Optimized for phones and tablets.

---

## 🚀 Getting Started

To run the app locally:

```bash
git clone https://github.com/raimonvibe/animalguesses-web.git
cd animalguesses-web
# Open index.html in your browser
```
